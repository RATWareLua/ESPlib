# ESPlib

Тестовая расширяемая ESP либа, можно дописывать любой кал который прийдёт в голову, всё что захочется отрендерить через Drawing.

---

## Совместимость
Тестилось сие чудо на ебаном потазиуме у которого собственно ужаснейшая Drawing Lib из всех +- адекватных екзеков, от эксплоита к эксплоиту оно всё отличается по-этому прийдётся тестить на месте. 

Для этого есть:
* `exampleLoader.luau` — 2д корнеры с трассером тимчеком и тимколором.
* `exampleLoader3D.luau` — аналогичный лоадер, но с 3д корнер боксами.

## Как использовать
В `USAGE.md` подробно описано как юзать эту либу (клопус постарался), начиная с того как писать модули заканчивая тем как юзать контракты и API твоего-же модуля в лоадере, если мозг не сгнил сможешь кинуть доку в чат с ЛЛМ и попросить сделать то что тебя интересует, главное описать нормально и не давать эту задачу гемини до момента пока гугл не выпустит обновлённую адекватную версию.
